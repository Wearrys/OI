# 主要目标

OI考试的最主要目标不是通过的题目数, 而是在考试规定的时间限制内取得尽可能高的分数.

达成目标的努力有几个最基本的方面:

- 保证稳定获得的分数不丢掉, 这建立在对题目意思的正确理解上, 同时也需要合理分配考试时间.

- 选择适合的做题顺序, 尽量使得得分的效率比较高, 在一道题目高分和满分实现难度过大时可以考虑舍弃一部分的分数换取其他题目高分.

- 注重细节上的成败, 不犯一些比较低级的错误, 觉得正确的题目尽可能留时间对拍并认真检查边界条件.

- 要有梦想, 对于不会做的题或者来不及写的题或者部分分, 不要什么都不写.

# 正确心态

首先不要以功利作为主要目标, 迷失自我的时候想想学习OI的初衷, 得之淡然, 失之坦然.

不急于去写一个有一点想法的题目, 尽可能地完善想法再实现程序, 警惕可能出错的部分, 保持考试大局观.

# 应对特殊情况

考试中遇到计划外的情况, 以题目较难等为例时, 尽量稳住心态, 选择比较好下手的题目去做, 读题的时候要考虑清楚每一道题目的稍进阶暴力写法,对于一些结论性的题目, 注意验证结论的正确性, 最好确定算法正确性之后再开始写题.
